## 二、射击手感的核心：鼠标灵敏度系统

### 2.1 基础公式

```
鼠标灵敏度公式：
真实旋转角度 = 鼠标移动距离(px) × 灵敏度系数 × 游戏引擎yaw值

cm/360 = (360 × 2.54) / (DPI × 游戏内灵敏度 × 游戏yaw值)
eDPI = DPI × 游戏内灵敏度  ← 仅限同款游戏内比较
```

### 2.2 Three.js 中的实现

**官方 PointerLockControls 的灵敏度模型：**
```javascript
// Three.js 内部常量
const _MOUSE_SENSITIVITY = 0.002;  // 像素→弧度转换基准

// 每帧旋转计算
_euler.y -= event.movementX * _MOUSE_SENSITIVITY * this.pointerSpeed;
_euler.x -= event.movementY * _MOUSE_SENSITIVITY * this.pointerSpeed;

// 用户可调参数
controls.pointerSpeed = 1.0;  // 默认，范围建议 0.1 ~ 5.0
controls.minPolarAngle = 0;     // 向上看限制
controls.maxPolarAngle = Math.PI; // 向下看限制
```

**关键参数解析：**

| 参数 | 默认值 | 含义 | 调校建议 |
|------|--------|------|----------|
| `_MOUSE_SENSITIVITY` | `0.002` | 像素→弧度基准转换 | 不修改，用pointerSpeed |
| `pointerSpeed` | `1.0` | 用户灵敏度倍数 | 0.5=慢/精确，2.0=快/灵活 |
| `movementX` | 鼠标报告 | 水平移动像素(无边界) | 启用unadjustedMovement |
| `Euler Order` | `'YXZ'` | 先偏航后俯仰 | 不可改变，否则视角会翻滚 |

### 2.3 原始鼠标输入（关键！）

```javascript
// 必须禁用鼠标加速度才能获得一致的FPS手感
canvas.requestPointerLock({ unadjustedMovement: true });

// 检查是否成功
document.addEventListener('pointerlockchange', () => {
  if (document.pointerLockElement === canvas) {
    console.log('Pointer locked — raw input active');
  }
});
```

**为什么这至关重要：**
- 没有 `unadjustedMovement`，Windows的"提高指针精确度"会扭曲鼠标数据
- 同样的物理移动距离产生不同的旋转角度 → 肌肉记忆无法建立
- 所有专业FPS训练器都强制使用原始输入

### 2.4 灵敏度转换系统

```
要从游戏A转到游戏B（保持相同的肌肉记忆）：
目标灵敏度 = 源灵敏度 × (源游戏yaw / 目标游戏yaw)

各游戏Yaw值参考：
CS2/Apex:      0.022
Valorant:      0.07
Overwatch 2:   0.0066
Fortnite:      0.005555
CoD Warzone:   0.0066
R6 Siege:      0.00572958
```

**在训练器中实现灵敏度导入：**
```typescript

function calculatePointerSpeed(preset: SensitivityPreset): number {
  const cmPer360 = (360 * 2.54) / (preset.userDPI * preset.userSensitivity * preset.yawValue);

  // 需要实测校准，因为浏览器鼠标报告与操作系统交互有微妙差异
```

```typescript
class HitscanSystem {
  private raycaster: THREE.Raycaster;

  shoot(): HitResult | null {
    // 从相机中心发射射线（模拟准星位置）
    this.raycaster.setFromCamera(
      new THREE.Vector2(0, 0),  // 屏幕中心 = 准星位置
      this.camera
    );

    const intersects = this.raycaster.intersectObjects(
      this.targetObjects,
      false  // 不递归子对象
    );

    if (intersects.length > 0) {
      const hit = intersects[0];
      return {
        point: hit.point.clone(),
        normal: hit.face.normal.clone(),
        distance: hit.distance,
        target: hit.object.userData.targetEntity,  // 关联到靶子实体
        isHeadshot: this.checkHeadshot(hit)
      };
    }
    return null;
  }
}
```

### 3.2 射速控制

```typescript
class FireRateController {
  private fireRateRPM: number;      // 每分钟射速
  private shotInterval: number;     // 毫秒间隔
  private lastShotTime: number = 0;
  private burstCount: number = 0;   // 当前连发计数
  private maxBurst: number;         // 最大连发数（0=无限制）
  private firingMode: 'semi' | 'burst' | 'auto';

  constructor(rpm: number, mode: 'semi' | 'burst' | 'auto') {
    this.fireRateRPM = rpm;
    this.shotInterval = 60000 / rpm;  // RPM → 毫秒
    this.firingMode = mode;
  }

  canFire(now: number, triggerDown: boolean): boolean {
    // 半自动：每按一次扳机一发
    if (this.firingMode === 'semi') {
      if (!triggerDown || this.triggerReleased) return false;
      this.triggerReleased = false;
    }
    // 时间检查
    if (now - this.lastShotTime < this.shotInterval) return false;
    // 连发限制
    if (this.maxBurst > 0 && this.burstCount >= this.maxBurst) return false;

    this.lastShotTime = now;
    this.burstCount++;
    return true;
  }

  resetBurst() { this.burstCount = 0; }
}
```

### 3.3 后坐力系统（核心手感）

```
后坐力 ≠ 散布

后坐力 (Recoil)：可预测的、可学习的枪口上扬模式
散布 (Spread)：  随机的弹道偏离，不可控
```

### 3.4 弹道散布

```typescript
class SpreadController {
  private baseSpread: number;      // 基础散布角(度)
  private spreadPerShot: number;   // 每发增加的散布
  private maxSpread: number;       // 最大散布
  private currentSpread: number;   // 当前散布
  private recoveryPerSec: number;  // 每秒回复量

  getSpreadAngle(): number {
    // 当前散布 = 基础 + 累积 - 时间回复
    return this.currentSpread;
  }

  applyShotSpread(direction: Vector3): Vector3 {
    // 在当前散布锥内随机偏移
    const angle = (Math.random() * 2 - 1) * this.currentSpread;
    const axis = new Vector3(
      Math.random() * 2 - 1,
      Math.random() * 2 - 1,
      0
    ).normalize();
    return direction.applyAxisAngle(axis, angle * Math.PI / 180);
  }

  onShot() {
    this.currentSpread = Math.min(
      this.maxSpread,
      this.currentSpread + this.spreadPerShot
    );
  }

  update(dt: number) {
    this.currentSpread = Math.max(
      this.baseSpread,
      this.currentSpread - this.recoveryPerSec * dt
    );
  }
}
```

## 四、命中反馈系统

### 4.1 视觉反馈层次

```
第一层：准星反馈 (< 50ms)
  └─ 命中准星扩大 + 颜色变化

第二层：靶子反馈 (0ms)
  └─ 靶子闪烁/破碎/消失 + 击中标记

第三层：HUD反馈 (< 100ms)
  └─ 伤害数字 + 得分动画 + 连击计数

第四层：屏幕反馈 (同步)
  └─ 命中标记 + 轻微屏幕震动 + 边框闪烁

第五层：环境反馈 (持续)
  └─ 弹孔贴花 + 弹道线 + 粒子效果
```

**实现示例：**
```typescript
class HitFeedbackSystem {
  // 准星反馈
  flashCrosshair(hit: HitResult) {
    this.crosshair.setColor(hit.isHeadshot ? '#FF0000' : '#FFFFFF');
    this.crosshair.setScale(1.3);  // 扩大30%
    this.crosshair.fadeTo(1.0, 200); // 200ms内恢复到正常大小
  }

  // 命中标记
  showHitmarker(hit: HitResult) {
    const marker = this.hitmarkerPool.get();
    marker.show(hit.isHeadshot ? 'headshot' : 'body');
    marker.fadeOut(400);  // 400ms淡出
  }

  // 伤害数字
  showDamageNumber(hit: HitResult, damage: number) {
    const num = this.damageNumPool.get();
    num.text = `${damage}`;
    num.position = hit.point;
    num.color = hit.isHeadshot ? '#FFD700' : '#FF4444';
    num.animate({ y: '+20', alpha: 0, duration: 800 });
  }

  // 靶子反馈
  animateTarget(target: Target, hit: HitResult) {
    target.playAnimation(hit.isHeadshot ? 'head_hit' : 'body_hit');
    target.spawnParticles(hit.point, hit.isHeadshot ? 'spark' : 'blood');
  }

  // 屏幕效果
  applyScreenEffects(damage: number) {
    // 击杀时的微妙屏幕震动
    if (damage >= 100) {
      this.cameraShake.addTrauma(0.3);
      this.timeScale.hitPause(0.05);  // 50ms 命中暂停
    }
  }
}
```

### 4.2 音效反馈设计

```
射击音效链：
  扣扳机 → 枪声(带随机变体) → 弹壳掉落
  命中   → 击中声(头部/身体不同) → 击杀确认音
  未命中 → 子弹擦过声(可选)
  换弹   → 弹匣弹出 → 弹匣插入 → 上膛

音效设计原则：
1. 枪声必须立即播放(< 10ms延迟)
2. 每把武器准备3-5个枪声变体(避免听觉疲劳)
3. 命中声在枪声后半拍(模拟子弹飞行时间)
4. 远处枪声加低通滤波(距离感)
5. 纯音调反馈命中和未命中 (击中的音调更高)
```

```typescript
// 命中暂停 — 微小的冻结帧制造"重量感"
class HitPause {
  trigger(duration: number = 0.05) {
    this.timeScale = 0.1;  // 几乎冻结
    this.remaining = duration;
  }

  update(dt: number) {
    if (this.remaining > 0) {
      this.remaining -= dt;
      if (this.remaining <= 0) {
        this.timeScale = 1.0;  // 恢复正常
      }
    }
  }
}

// 屏幕震动 — 创伤衰减模型
class CameraShake {
  trauma: number = 0;  // 0~1

  addTrauma(amount: number) {
    this.trauma = Math.min(1, this.trauma + amount);
  }

  update(dt: number): Vector3 {
    // 创伤平方衰减 → 震动非线性
    const shake = this.trauma * this.trauma;
    const offset = new Vector3(
      (Math.random() * 2 - 1) * shake * 0.05,
      (Math.random() * 2 - 1) * shake * 0.05,
      0
    );
    this.trauma = Math.max(0, this.trauma - dt * 1.5);  // 1.5/秒衰减
    return offset;
  }
}
```

```typescript
class TargetSpawner {
  private spawnArea: { width: number; height: number }; // 生成区域
  private padding: number = 0.1;  // 边缘留白(10%)
  private maxActive: number;       // 最大同时存在数

  getSpawnPosition(existingTargets: Target[]): Vector3 {
    // 避免与现有靶子重叠
    let pos: Vector3;
    let attempts = 0;
    const minDistance = targetSize * 3;

    do {
      pos = new Vector3(
        (Math.random() * 2 - 1) * this.spawnArea.width * (1 - this.padding),
        (Math.random() * 2 - 1) * this.spawnArea.height * (1 - this.padding),
        this.spawnDistance  // 固定深度
      );
      attempts++;
    } while (
      attempts < 10 &&
      existingTargets.some(t => t.position.distanceTo(pos) < minDistance)
    );

    return pos;
  }
```

### 5.3 移动靶子（追踪训练）

```typescript
type MovementPattern = 'linear' | 'sinusoidal' | 'random_walk' | 'bounce';

class TargetMovement {
  private pattern: MovementPattern;
  private speed: number;
  private direction: Vector3;
  private bounds: { min: Vector3; max: Vector3 };
  private changeTimer: number;
  private amplitude: number;  // 正弦波振幅
  private frequency: number;  // 正弦波频率
  private phase: number;      // 当前相位

  update(dt: number): Vector3 {
    switch (this.pattern) {
      case 'linear':
        return this.updateLinear(dt);
      case 'sinusoidal':
        return this.updateSinusoidal(dt);
      case 'random_walk':
        return this.updateRandomWalk(dt);
      case 'bounce':
        return this.updateBounce(dt);
    }
  }

  private updateSinusoidal(dt: number): Vector3 {
    this.phase += this.frequency * dt;
    return new Vector3(
      Math.sin(this.phase) * this.amplitude,
      Math.cos(this.phase * 1.3) * this.amplitude * 0.5,
      0
    );
  }

  private updateRandomWalk(dt: number): Vector3 {
    this.changeTimer -= dt;
    if (this.changeTimer <= 0) {
      // 每0.5-1.5秒改变方向
      this.direction = new Vector3(
        (Math.random() * 2 - 1),
        (Math.random() * 2 - 1),
        0
      ).normalize();
      this.changeTimer = 0.5 + Math.random() * 1.0;
    }
    const nextPos = this.position.clone().add(
      this.direction.clone().multiplyScalar(this.speed * dt)
    );
    // 边界弹跳
    if (nextPos.x < this.bounds.min.x || nextPos.x > this.bounds.max.x)
      this.direction.x *= -1;
    if (nextPos.y < this.bounds.min.y || nextPos.y > this.bounds.max.y)
      this.direction.y *= -1;
    return nextPos;
  }
}
```

## 六、计分与统计系统

### 6.1 计分公式

```
基础命中分数 = 靶子基础分 × 距离倍率 × 头部倍率 × 连击倍率

距离倍率：
  close (0-15m):   ×0.7  (太容易了)
  medium (15-30m): ×1.0  (基准)
  far (30-60m):    ×1.5  (有挑战)
  extreme (60m+):  ×2.5  (精英射手奖励)

连击倍率：
  streak × 0.1 (最小1.0，无上限)

未命中惩罚：
  扣分 = 靶子基础分 × 0.2 (仅扣20%，避免打击信心)
```

```typescript
class HitHeatmap {
  private grid: number[][];  // 命中计数矩阵
  private resolution: number; // 网格分辨率

  recordHit(screenX: number, screenY: number): void {
    const gx = Math.floor(screenX / this.resolution);
    const gy = Math.floor(screenY / this.resolution);
    this.grid[gy][gx]++;
  }

  render(ctx: CanvasRenderingContext2D): void {
    // 冷区=蓝色边缘 → 提示周边视野训练不足
    // 热区=红色中心 → 确认中心瞄准稳定
    const maxHits = this.getMaxHits();
    for (let y = 0; y < this.grid.length; y++) {
      for (let x = 0; x < this.grid[y].length; x++) {
        const intensity = this.grid[y][x] / maxHits;
        ctx.fillStyle = this.heatColor(intensity);
        ctx.fillRect(x * this.resolution, y * this.resolution,
                     this.resolution, this.resolution);
      }
    }
  }
}
```

### 7.2 动态准星

```typescript
class DynamicCrosshair {
  private config: CrosshairConfig;
  private currentGap: number;

  update(playerState: PlayerState, weaponState: WeaponState): void {
    // 间隙受多种因素影响
    let gapMultiplier = 1.0;

    // 移动影响散布
    if (playerState.isMoving) gapMultiplier *= 1.8;
    if (playerState.isSprinting) gapMultiplier *= 3.0;
    if (playerState.isCrouching) gapMultiplier *= 0.6;
    if (playerState.isAirborne) gapMultiplier *= 4.0;

    // 射击影响散布
    gapMultiplier += weaponState.spreadAngle * 0.5;

    // 平滑过渡
    this.currentGap = lerp(this.currentGap,
      this.config.gap * gapMultiplier, 10 * deltaTime);
  }

  render(ctx: CanvasRenderingContext2D): void {
    const gap = this.currentGap;
    const size = this.config.size;

    // 十字线四臂
    ctx.strokeStyle = this.config.color;
    ctx.lineWidth = this.config.thickness;

    // 上
    ctx.beginPath();
    ctx.moveTo(centerX, centerY - gap);
    ctx.lineTo(centerX, centerY - gap - size);
    ctx.stroke();
    // 下、左、右同理
  }
}
```

### 10.2 核心循环架构

```typescript
class TrainingRangeGame {
  update(): void {
    const dt = this.time.getDelta();

    // 第1步：输入处理
    this.inputManager.update();
    this.sensitivityManager.apply(this.inputManager.mouseDelta);

    // 第2步：相机更新(被输入驱动)
    this.fpsCamera.update(dt);

    // 第3步：场景更新
    this.scenarioManager.update(dt);
    this.targetSpawner.update(dt);
    for (const target of this.activeTargets) {
      target.update(dt);
    }

    // 第4步：射击处理(如果扳机按下)
    if (this.inputManager.isTriggerDown) {
      this.handleShooting(dt);
    }

    // 第5步：反馈效果处理
    this.cameraShake.update(dt);
    this.hitPause.update(dt);
    this.hitFeedback.update(dt);
    this.crosshair.update(dt, this.playerState, this.currentWeapon);

    // 第6步：统计更新
    this.scoreManager.update(dt);
    this.sessionStats.update(dt);

    // 第7步：渲染
    this.renderer.render(this.scene, this.camera.getCamera());

    // 第8步：性能监控
    this.stats.update();
  }
}
```

## 十二、关键技术决策总结

| 决策点 | 推荐方案 | 核心理由 |
|--------|----------|----------|
| 瞄准输入 | PointerLock + unadjustedMovement | 原始鼠标数据是FPS手感的基础 |
| 灵敏度模型 | 0.002基准 + pointerSpeed倍数 | 与Three.js生态一致，支持游戏间转换 |
| 射击检测 | 屏幕中心射线检测 | 训练器不需要弹道物理，射线最快 |
| 后坐力 | 确定性的点序列模式 | 可学习、可掌握 |
| 散布 | 锥角随机 + 时间回复 | 提供可控随机性 |
| 命中反馈 | 五层同步反馈栈 | 即时、分层、多感官 |
| 靶子管理 | 对象池 | 高性能复用，减少GC |
| 计分系统 | 基础分×倍率连击 | 简单透明，激励精度 |
| 训练模式 | Scenario模板 | JSON配置化，易于扩展 |
| 音效 | 多变体池 + AudioBuffer | 避免听觉疲劳 |
